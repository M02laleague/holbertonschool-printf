##### Command for compilation
``"gcc -Wall -Werror -Wextra -pedantic -std=gnu89"``

Here is the requirement you will need to use our function _printf :

- Ubuntu 20.04 LTS using gcc
- Compilation flags: -Wall -Werror -Wextra -pedantic -std=gnu89
- Code should use the Betty style


## Description

*The printf function sends formatted output to stdout.
A custom _printf() for learning purposes was developed by cohort #24 students DIALLO & GEMICI.
_printf() function format string is a character string, beginning and ending whith char, if any. 
These arguments are placed using the percentage '%' operator.
Available conversion specifiers are: c, s, d, i, b, u, o, x, X, p, r, R, %.*
